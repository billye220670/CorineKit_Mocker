# MockerMix - 音频混音播放器

一个基于Web Audio API的音频混音播放器，支持随机播放、多音频混音、混响效果和多语言界面。

## 功能特性

- **音频管理**：拖拽音频文件或文件夹添加到播放列表
- **随机播放**：支持权重设置，可控制各音频的播放概率
- **音频混音**：最多支持6个音频同时混音播放
- **混响效果**：可调节混响强度，为音频添加空间感
- **播放控制**：
  - 基本间隔时间调节
  - 随机偏移百分比
  - 防止重复播放
  - 两种触发模式（定时器/音频结束）
- **主题切换**：支持深色/亮色主题
- **多语言**：支持中文、英文、日文界面
- **设置保存**：自动播放保存设置到本地存储

## 使用方法

直接用浏览器打开 `Index.html` 即可使用。

### 快速开始

1. 将音频文件拖拽到页面顶部的拖拽区域
2. 点击「开始随机播放」按钮
3. 在控制面板调节各项参数

### 混音设置

- 混音数量设为 `0`：单音频顺序播放
- 混音数量设为 `1-6`：多音频同时混音播放

## 技术栈

- HTML5 + CSS3
- Tailwind CSS（样式框架）
- 原生 JavaScript
- Web Audio API（音频处理）
