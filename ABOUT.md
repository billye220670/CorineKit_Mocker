# 关于 MockerMix

## 项目介绍

MockerMix 是一款轻量级的Web音频混音播放器，无需安装任何软件，直接在浏览器中运行。适合用于背景音乐播放、音频测试、现场混音等场景。

## 主要功能

### 1. 播放列表管理
- 支持拖拽单个音频文件或整个文件夹
- 支持常见的音频格式（mp3、wav、ogg、m4a等）
- 实时显示播放列表，可点击播放单个音频
- 可设置每个音频的播放权重

### 2. 随机播放引擎
- 基于权重的随机选择算法
- 可调节基本播放间隔（0.1秒 - 4秒）
- 支持随机偏移，增加播放变化性
- 可防止连续重复播放同一音频

### 3. 混音功能
- 支持最多6个音频同时混音
- 每路音频独立音量控制
- 混音模式下自动管理播放资源

### 4. 音频效果
- **混响效果**：使用卷积混响技术模拟空间感
- **湿/干信号混合**：可自由调节混响强度

### 5. 用户界面
- **深色/亮色主题**：一键切换
- **多语言支持**：中文、英文、日文
- **响应式设计**：适配各种屏幕尺寸
- **本地存储**：自动保存播放设置

## 使用场景

- 直播间的背景音乐轮播
- 播客节目的音乐过渡
- 零售店铺的背景音乐播放
- 音频素材的随机预览
- 现场活动的音效管理

## 技术特点

- 纯前端实现，无需后端服务
- 使用 Web Audio API 进行专业音频处理
- 响应迅速，交互流畅
- 代码简洁，易于二次开发
